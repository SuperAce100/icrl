## Schema

```python
class Trajectory(BaseModel):
    id: str  # uuid default
    goal: str
    plan: str
    steps: list[Step]
    success: bool
    metadata: dict[str, Any] = {}
```

## Utility

```python
trajectory.to_example_string() -> str
```

Formats the trajectory into an in-context example block.
