## Overview

The ICRL TypeScript package (`icrl`) provides the same trajectory-learning framework as the Python package. Use the pages in this section for detailed API documentation.

## Core Components

- [Agent](/api-reference/typescript-agent) — main agent class
- [TrajectoryDatabase](/api-reference/typescript-trajectory-database) — trajectory store with semantic search
- [TrajectoryRetriever](/api-reference/typescript-trajectory-retriever) — step-level retrieval
- [CurationManager](/api-reference/typescript-curation-manager) — automatic curation
- [ReActLoop](/api-reference/typescript-react-loop) — planning, reasoning, acting loop

## Storage & Protocols

- [StorageAdapter](/api-reference/typescript-storage) — pluggable storage interface; built-in `FileSystemAdapter`
- [Environment](/api-reference/typescript-environment) — environment protocol
- [LLMProvider](/api-reference/typescript-llm-provider) — LLM protocol
- [Embedder](/api-reference/typescript-embedder) — embedding protocol

## Models

- [Models](/api-reference/typescript-models) — `Trajectory`, `Step`, `Message`, `StepExample`, helpers

## Providers

- [OpenAI Provider](/api-reference/typescript-openai-provider) — `OpenAIProvider`, `OpenAIEmbedder`
- [Anthropic Provider](/api-reference/typescript-anthropic-provider) — `AnthropicProvider`, `AnthropicVertexProvider`
