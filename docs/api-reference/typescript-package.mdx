## Agent

```ts
new Agent(options: AgentOptions)
await agent.init()
await agent.train(env, goal)
await agent.run(env, goal)
await agent.trainBatch(envFactory, goals)
await agent.runBatch(envFactory, goals)
agent.getDatabase()
agent.getStats()
```

`AgentOptions` includes:

- `llm`, `storage`, `embedder`
- `planPrompt`, `reasonPrompt`, `actPrompt`
- `k`, `maxSteps`
- `seedTrajectories`, `onStep`
- `curationThreshold`, `curationMinRetrievals`
- `verifyTrajectory`

## Database and Retrieval

- `TrajectoryDatabase(storage, embedder, options?)`
- `TrajectoryRetriever(database, k?)`
- `CurationManager(database, options?)`
- `ReActLoop(llm, retriever, options)`
