## Interface

```ts
interface Embedder {
  readonly dimension: number;
  embed(texts: string[]): Promise<number[][]>;
  embedSingle(text: string): Promise<number[]>;
}
```

Used by `TrajectoryDatabase` for semantic similarity search. `dimension` is the embedding vector size.
