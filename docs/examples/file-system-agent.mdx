## Source Files

- `examples/test_with_mock.py`
- `examples/demo_with_real_llm.py`

## Run With Mock LLM (no API keys)

```bash
uv run python examples/test_with_mock.py
```

This demonstrates:

- training and inference flows
- retrieval usage
- persistent trajectory loading

## Run With Real Model

```bash
OPENAI_API_KEY=... uv run python examples/demo_with_real_llm.py
```

Optional model override:

```bash
MODEL=gpt-4o-mini uv run python examples/demo_with_real_llm.py
```

## Environment Behavior

`FileSystemEnvironment` supports command-like actions such as:

- `ls`
- `cd`
- `cat`
- `find`
- `pwd`
- `mkdir`
- `cp`

Each task has a verification function that determines `success`.
