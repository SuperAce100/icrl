## Source Files

- `examples/harbor_coding_agent.py`
- `examples/test_harbor_coding.py`

## What It Demonstrates

- coding workspace simulation (`/workspace/src`, `/workspace/tests`)
- command-driven code editing/navigation
- baseline vs post-training behavior
- verifiable coding tasks (analysis/debugging/testing categories)

## Run Demo

```bash
OPENAI_API_KEY=... uv run python examples/harbor_coding_agent.py
```

## Run Tests

```bash
uv run pytest examples/test_harbor_coding.py -v
```

## Task Structure

Tasks are `CodingTask` objects with:

- `goal`
- `verify(workspace_state) -> bool`
- optional setup hooks
- metadata (`difficulty`, `category`)

This makes it suitable for repeatable benchmarking-style workflows.
