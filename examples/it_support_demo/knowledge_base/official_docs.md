# ACME Corp IT Documentation

> Last updated: 18 months ago

## VPN Access

To connect to the corporate VPN:
1. Download GlobalProtect from the Software Center
2. Enter server address: `vpn.acme.com`
3. Log in with your ACME credentials
4. Click Connect

For VPN issues, please contact the IT Help Desk.

## Email Setup

### Desktop (Outlook)
Outlook should auto-configure when you sign in with your ACME account.

### Mobile
1. Download Microsoft Outlook from your app store
2. Add your ACME email account
3. Complete the sign-in process

For email issues, contact support@acme.com.

## Network Access

ACME Corp provides two WiFi networks:
- **ACME-Corp**: For employees (requires login)
- **ACME-Guest**: For visitors

## Software Requests

To request new software:
1. Log into ServiceNow
2. Navigate to Service Catalog
3. Submit a Software Request form

Allow 3-5 business days for processing.

## GitHub Access

All engineers should have GitHub access through ACME's organization.
1. Ensure you've accepted the org invitation
2. Clone repos using HTTPS or SSH
3. For access issues, contact your team lead

## Database Access

Database access is managed through the DBA team.
- Submit a request in ServiceNow
- Include business justification
- Access is reviewed quarterly

## Common Issues

### "I forgot my password"
Use the self-service password reset at https://password.acme.com

### "My laptop is slow"
Try restarting. If issues persist, contact IT.

### "I can't access a website"
Some sites are blocked by corporate policy. Submit an exception request if needed for work.

---

*For all other issues, please submit a ticket in ServiceNow or email helpdesk@acme.com*
